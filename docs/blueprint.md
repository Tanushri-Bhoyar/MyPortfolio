# **App Name**: DevCanvas

## Core Features:

- Personal Introduction Section: A dynamic hero section to prominently display your name, role, and a compelling introduction, inspired by leading portfolio designs.
- Contact Information Display: Dedicated section for clear display of contact details: email, phone, and LinkedIn, complete with appropriate visual icons for easy recognition.
- Dynamic Skills Listing: A modular component to showcase your proficiency in technologies like Java, JavaScript, React, and Python, easily extendable through props to include future skills.
- Prop-Driven Project Showcase: A versatile section to feature projects such as the Amazon clone, invoice manager, and ice cream parlor website, with data updated simply by passing new props to the component.
- Responsive Layout Integration: Utilize Bootstrap for a fully responsive design, ensuring the portfolio looks and functions seamlessly across all device sizes from mobile to desktop.
- Modular UI Architecture: Develop all user interface components as separate, prop-driven modules, allowing for easy updates and maintenance of content like experience, technologies, and projects without code changes.
- AI-Powered Content Refinement Tool: An internal tool to assist you in generating or refining impactful descriptions for your 'About Me' section or project summaries, leveraging generative AI to enhance clarity and engagement.

## Style Guidelines:

- Background Color: A very subtle and light violet (#F9F7FC) to establish a clean and elegant base for the 'light violet' theme.
- Primary Color: A medium, rich violet (#6633CC) used for key elements like headers, main text, and interactive components to provide depth and align with the 'light violet' aesthetic.
- Accent Color: A vibrant magenta-violet (#DB57DB) strategically employed for calls-to-action, highlights, and interactive states to draw attention and add a touch of modern flair.
- Headline Font: 'Space Grotesk' (sans-serif) for a modern, tech-savvy, and slightly distinctive look in section titles and main headings.
- Body Font: 'Inter' (sans-serif) for all body text, ensuring excellent readability, neutrality, and professionalism across the portfolio.
- Use modern, minimalist line icons for contact details (email, phone, LinkedIn) ensuring consistency and clear visual communication throughout the site.
- Responsive grid layout, leveraging Bootstrap to ensure optimal presentation and usability across all device screen sizes, from mobile to desktop, with content adapting gracefully.
- Subtle and elegant fade-in transitions for content as it loads, and gentle hover effects on interactive elements like project cards and navigation links, for a polished user experience.
